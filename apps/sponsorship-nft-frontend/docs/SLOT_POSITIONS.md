# 槽位位置说明

## 短裤图片规格

- **容器尺寸**: 400px × 500px
- **图片格式**: PNG（支持透明背景）
- **文件路径**: `public/shorts/shorts.png`

生成短裤图片时，请确保以下槽位位置正确：

### Slot A - 左腿上
- **X**: 80px
- **Y**: 120px
- **Width**: 100px
- **Height**: 40px

### Slot B - 左腿下
- **X**: 80px
- **Y**: 280px
- **Width**: 100px
- **Height**: 40px

### Slot C - 右腿上
- **X**: 220px
- **Y**: 120px
- **Width**: 100px

### Slot D - 右腿下
- **X**: 220px

## 注意事项

1. 槽位区域应该留空或使用浅色占位符
2. Logo 将动态覆盖在槽位区域上
3. 确保槽位区域有足够的对比度以显示 Logo
4. 每个槽位尺寸为 100px × 40px（横条 Logo 适合的比例）
