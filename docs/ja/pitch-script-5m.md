# InFight AMM ピッチスクリプト（5 分版）

## Intro（イントロ）

Gm Guys、Kyohei です。

最初に質問です。
皆さん、格闘技はお好きでしょうか？
僕自身も大の格闘技ファンです。

格闘技は世界的に巨大なマーケットですが、オンラインのファン体験は今も “観戦するだけ”。
試合の興奮にリアルタイムで参加する方法は存在しません。

そこで私たちは InFight AMM を開発しています。
世界初の、格闘技に特化したリアルタイム予測市場です。

“格闘技 × Web3” を両方深く理解している、非常にユニークなチームです。

## Why InFight?（なぜ試合中なのか）

私たちはまず、格闘技 × Web3 のユースケースを実際の業界知見にもとづきフェーズごとに整理しました。

その中で最も重要なのが 試合中のリアルタイム予測市場です。

理由は 3 つあります。

まず、ファンが最も興奮するのは “ライブ中”。
次に、1 大会に複数試合があるため、繰り返し参加できます。
そして何より、世界にまだ “格闘技特化のリアルタイム予測市場” が存在しない。

ここに圧倒的な市場ギャップがあります。

## Existing Problems（既存予測市場の限界）

Polymarket のような既存の予測市場は、格闘技には向いていません。

トレーダー向けに高度化しすぎていて、一般ファンには複雑すぎる。
さらに、決済に数日かかるため、1 試合目でロックされた資金を 2 試合目に使えない。
また、パンチ数や支配時間といったライブデータを秒単位で反映できず、格闘技のスピードに追いつけません。

だからこそ、“高速・直感的・格闘技特化” の新しい市場が必要です。

## Solution（解決策）

そこで InFight AMM です。

UI はスポーツファン向けに設計され、ライブ配信と統合されているため、誰でも直感的に使えます。

AMM は LSMR モデルを採用し、試合の流れに合わせてオッズがリアルタイムで反応します。
モメンタムが変われば、オッズも即座に変わる。まるで選手と戦っているような臨場感です。

そして 最大の強みが “即時決済” です。
従来は 3〜7 日かかっていた決済が、InFight AMM では 試合終了後わずか 10 分。
そのまま次の試合に参加できます。
格闘技のライブ体験と完全にフィットします。

次に強力なのが、Nautilus によるリアルタイム・完全検証済みデータです。
ONE Championship や UFC Stats の公式データを、Nautilus が安全なエンクレーブ内で暗号署名し、
支配時間・打撃数・テイクダウン成功率などを リアルタイムかつトラストレス にオンチェーンへ送信します。

さらに、すべてが Sui 上で動作します。
サブセカンド確定と 1 セント未満のガスで、数百万人規模のイベントにも対応できます。
