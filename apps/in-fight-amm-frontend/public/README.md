# Public Assets

## Required Files

### 1. Video File: `fight-video.mp4`

Place your fight video file in this folder with the name `fight-video.mp4`.

**Recommended Video Specifications:**
- Format: MP4
- Resolution: 1920x1080 (Full HD) or higher
- Codec: H.264
- File size: Under 50MB for optimal loading

### 2. Fighter Photos

Place fighter photos in this folder with the following names:

- `fighter-a.jpg` - Photo for Fighter A (Mike "Dragon")
- `fighter-b.jpg` - Photo for Fighter B (John "Thunder")

**Recommended Image Specifications:**
- Format: JPG or PNG
- Resolution: 400x400 pixels or higher (square format recommended)
- File size: Under 500KB each
- Face should be clearly visible and centered

### How to Add Files:

1. Copy your video file to this folder and rename it to `fight-video.mp4`
2. Copy fighter photos to this folder and rename them to `fighter-a.jpg` and `fighter-b.jpg`
3. Reload the application
