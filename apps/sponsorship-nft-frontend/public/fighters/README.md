# Fighter Images

Place fighter images in this directory.

## File Naming

- Fighter 1 (Weirui): `weirui.jpg` or `weirui.png`
- Add more fighters as needed

## Supported Formats

- JPG/JPEG
- PNG
- WebP

## Recommended Size

- Aspect ratio: 16:9 or 4:3
- Recommended: 800x600px or larger

## Usage

Update the `avatar` field in `data/mockData.ts`:

```typescript
{
  id: '1',
  name: 'Weirui',
  avatar: '/fighters/weirui.jpg', // or .png
  // ...
}
```
