# Event Posters Guide

This folder should contain event poster images for the homepage.

## Required Files

Place event poster images in this folder with the following names:

### Main Event Poster (Currently Live)
- `event-poster-1.jpg` - UFC 300: Mike "Dragon" vs John "Thunder"

### Upcoming Events
- `event-poster-2.jpg` - Boxing Championship
- `event-poster-3.jpg` - Kickboxing Grand Prix
- `event-poster-4.jpg` - Muay Thai Championship
- `event-poster-5.jpg` - ONE Championship

## Image Specifications

**Recommended:**
- Format: JPG or PNG
- Aspect Ratio: 16:9 or 3:2 (horizontal/landscape)
- Resolution: 1200x675px or higher
- File size: Under 1MB each
- High quality action shots or promotional posters

**For Hero Banner (event-poster-1.jpg):**
- This image will be displayed as a large hero banner (400px height)
- Should be high resolution and visually striking
- Will have a gradient overlay for text readability

**For Event Cards (event-poster-2 to 5):**
- Will be displayed at 192px height in cards
- Should clearly show fighters or event branding
- Will have category and live badges overlaid

## How to Add Files

1. Copy your event poster images to this folder
2. Rename them according to the list above (event-poster-1.jpg, event-poster-2.jpg, etc.)
3. Reload the application

## Tips for Best Results

- Use high-quality promotional materials or action shots
- Ensure main subjects (fighters) are clearly visible
- Avoid images with too much text overlay (as we add our own)
- Landscape orientation works best for the layout
- Consider the gradient overlay on the hero image when choosing
