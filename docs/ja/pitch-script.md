コミュニティ主導のマッチメイクや、選手スポンサー広告の NFT 化など、活用できる領域は非常に広い。

将来的にはこのフルエコシステムを構築していきますが、今日お話しするのは、その中でも最も重要な “試合中のリアルタイム予測市場” です。

ここに、大きな市場ギャップがあります。

## 予測市場の課題

Polymarket のような既存の予測市場は、格闘技には適していません。

まず、トレーダー向けに進化しすぎていて、一般ファンには複雑すぎます。

さらに致命的なのは、決済に数日かかる点です。
1 試合目で賭けた資金がロックされたままでは、2 試合目・3 試合目に参加できません。

そして何より、リアルタイムでポジションを高速に調整する用途に向いていません。
パンチ数や支配時間など、秒単位のライブデータを反映できないため、格闘技のスピード感に追いつけない。

だからこそ、“高速・直感的・格闘技特化” の予測市場が絶対に必要なのです。

## Solution（解決策）

そこで InFight AMM です。

UI はプロトレーダーではなく、一般のスポーツファンのために設計されています。
ライブ配信と予測市場の UI が統合されており、難しい知識がなくても直感的に使えます。

そして AMM は LSMR モデルを採用。
試合の流れに合わせて、オッズがリアルタイムで反応します。
モメンタムが変われば、オッズも即座に変わる。

### 最大の強み — “マーケットの即時決済”

従来の予測市場は決済に 3〜7 日かかりますが、InFight AMM は試合終了後わずか 10 分で勝利金を Claim できます。

つまり、そのまま次の試合に参加できます。
この “即時性” が、格闘技のライブ体験に完璧にフィットします。
まるで選手と一緒に戦っているかのような体験です。

### 次点の強み — Nautilus による完全検証済みのリアルタイムデータ

ONE Championship や UFC Stats の公式データを取得し、
Nautilus が安全なエンクレーブ内で暗号署名。

支配時間、打撃数、テイクダウン成功率などを、リアルタイム・完全検証済み・トラストレスでオンチェーンへ送信します。

### Sui が可能にする超高速パフォーマンス

Sui のサブセカンド確定、1 セント未満のガスにより、数百万人規模のイベントにも対応可能です。
